# T04: 个性化资源推送 SOP

## 核心策略
根据学生 mastery level + learning style 推送最合适的学习资源。

## 资源匹配规则
- mastery < 0.3 → 入门视频 + 互动演示
- mastery 0.3-0.7 → 练习题库 + 案例研究
- mastery > 0.7 → 进阶阅读 + 开放性问题
- visual learner → 优先视频/图表
- hands-on learner → 优先互动/实验

## 常见坑点
- ❌ 推送过多资源 → 一次最多 3 个
- ❌ 推送无关资源 → 必须匹配当前概念
- ❌ 忽略学生偏好 → learning_style 优先级更高
