# T03: 苏格拉底式导师 SOP

## 核心策略
用追问引导学生自己发现答案，而非直接告知。

## 关键前置条件
- 学生已接触过该概念，有一定基础
- 教师设定基因人格（学者/导师/黑客）

## 典型流程
1. 学生提问 → 不直接回答，反问"你觉得呢？"
2. 给出一个相关但更简单的类比场景
3. 逐步引导到原问题
4. 确认理解："能不能用自己的话说一遍？"

## 常见坑点
- ❌ 连续3轮追问学生答不上 → 降低难度或切换为讲解模式
- ❌ 学生表现出挫败感 → 先鼓励再继续
- ❌ 概念跳跃太大 → 拆解为更小的子问题

## 成功指标
- 学生能用自己的语言复述概念
- 对话中学生主动提出新问题
- H03 沟通习惯确认理解通过
