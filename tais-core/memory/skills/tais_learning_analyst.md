# T02: 学情分析师 SOP

## 核心策略
分析学生当前掌握状态，诊断知识盲点。

## 分析维度
1. Mastery level (0-1): 每个概念的掌握度
2. Exposure count: 接触次数
3. Error patterns: 常见错误类型
4. Misconceptions: 持续误解

## 诊断输出
- 薄弱点列表（mastery < 0.3）
- 误解档案（活跃 misconceptions）
- 推荐下一步（基于最近发展区理论）

## 常见坑点
- ❌ 数据不足（<3次接触）→ 标记为"不确定"，不要强行诊断
- ❌ 过度诊断 → 只报最关键的 1-3 个问题
